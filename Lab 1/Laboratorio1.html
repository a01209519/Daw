Descripciones

URL: son las siglas para localizador uniforme de recursos. Es la direccion que tiene una página para poder ser localizada.[1]
MIME: son las siglas para multipurpose internet mail extensions. Son las extensiones que normalmente se usan para mandar archivos a travez del internet. [2]
Codigos de errores: Son las anomalías que pueden ocurrir en las transmiciones en linea.
HTTP: son las siglas para hypertext transfer protocol. Es el protocolo con el que el WWW define como los mensjaes deben de ser mandados y como atuar con respecto a ciertos comandos. [3]
HTML5: es el lenguaje que usa la WWW. Es la quinta revision, razon por la que es llamada html5.
JavaScript: Es un lenguaje que te permite relizar programas en web.
PHP: lenguaje para desarollar contenido de tipo dinámico. [4]
JavaServerPages: al igual que php sirve para hacer contenido dinamico pero en este caso se usa el lenguaje java.
Ajax: usa la combinacion de un buscador en HTMLHttp y Javascript y html paar desplegar la informacion. [5]
