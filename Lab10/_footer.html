Preguntas

¿Por qué es una buena práctica separar el controlador de la vista?
Es una medida de seguridad para que la gente que se meta al código solo vea el controlador y no pueda modificar la vista. Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?
Cookie: Una variable tipo array asociativo de variables pasadas al script actual a través de Cookies HTTP.
Request: Un array asociativo que por defecto contiene el contenido de $_GET, $_POST y $_COOKIE.

¿Qué es XSS y cómo se puede prevenir?
Es un ataque en el cual de injectan scripts a una página web de manera externa, estos scripts pueden realizar diferentes acciones, normalmente perjudiciales a la página. Esto se puede evitar con el comando htmlspecialchars.
